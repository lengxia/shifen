# 版本说明

该版本实现了基本的数据库连接，基本接口定义，工具类定义的工作。

作为basic版本，特此发布0.0.1

## 一、模块定义

模块定义定义当前的通用模块，现阶段包含通用api，通用模型，通用工具类，通用配置。

## 二、服务定义

现阶段实现了user服务登录功能，作为基本功能后期待完善
